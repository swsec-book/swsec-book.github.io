# Welcome

Questo sito web affianca il libro di testo *"Software Security. Metodi di analisi e mitigazione delle vulnerabilità"*, di prossima pubblicazione dalla casa editrice Pearson Italia.

Il sito contiene esempi pratici dei concetti trattati nel libro, con delle guide passo-passo per eseguirli, ed esercizi aperti per il lettore.

Il sito fornisce la possibilità di scaricare una macchina virtuale Linux, preconfigurata con tutti gli strumenti necessari per seguire gli esempi e gli esercizi proposti.

Per suggerimenti o per segnalare errori, è possibile contattare l'autore all'indirizzo email *roberto.natella at gssi.it*.
